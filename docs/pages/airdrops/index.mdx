# Get Airdrops

This page documents the API endpoint for retrieving Artist airdrop information on the In Process protocol.

## Endpoint

```http
GET https://inprocess.world/api/airdrops
```

## Query Parameters

The following query parameters are available. Only `artist_address` is required:

### Required query parameters

- `artist_address` (string): Artist address

### Optional query parameters

- `chainId` (number): Chain ID (default: Base, 8453)
- `offset` (number): Pagination offset (default: 0)

## Example Requests

:::code-group

```bash [cURL]
curl "https://inprocess.world/api/airdrops?artist_address=0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E&chainId=84532&offset=0"
```

```python [Python]
import requests

url = "https://inprocess.world/api/airdrops"
params = {
  "artist_address": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
  "chainId": 84532,
  "offset": 0
}
response = requests.get(url, params=params)
print(response.json())
```

```javascript [JavaScript]
const params = new URLSearchParams({
  artist_address: "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
  chainId: "84532",
  offset: "0",
});

const response = await fetch(`https://inprocess.world/api/airdrops?${params}`);
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface AirdropQueryParams {
  artist_address: string;
  chainId?: number;
  offset?: number;
}

const params: AirdropQueryParams = {
  artist_address: "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
  chainId: 84532,
  offset: 0,
};

const queryString = new URLSearchParams({
  artist_address: params.artist_address,
  chainId: (params.chainId ?? 8453).toString(),
  offset: params.offset?.toString() || "0",
});

const response = await fetch(
  `https://inprocess.world/api/airdrops?${queryString}`
);
const data = await response.json();
```
