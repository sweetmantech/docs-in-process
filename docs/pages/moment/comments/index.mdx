# Get Comments

This page documents the API endpoint for retrieving Moment comments on the In Process protocol.

## Endpoint

```http
POST https://inprocess.fun/api/moment/comments
```

## Request Body

Send a JSON object with the following fields:

```json
{
  "moment": {
    "contractAddress": string, // Moment contract address
    "tokenId": "1" // TokenId 
  },
  "chainId": number, // Chain ID
  "offset": 0 // Pagination offset
}
```

## Example Requests

:::code-group

```bash [cURL]
curl -X POST https://inprocess.fun/api/moment/comments \
  -H "Content-Type: application/json" \
  -d '{
    "moment": {
      "contractAddress": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
      "tokenId": "1"
    },
    "chainId": 84532,
    "offset": 0
  }'
```

```python [Python]
import requests

url = "https://inprocess.fun/api/moment/comments"
payload = {
  "moment": {
    "contractAddress": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
    "tokenId": "1"
  },
  "chainId": 84532,
  "offset": 0
}
response = requests.post(url, json=payload)
print(response.json())
```

```javascript [JavaScript]
const response = await fetch("https://inprocess.fun/api/moment/comments", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    "moment": {
      "contractAddress": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
      "tokenId": "1"
    },
    "chainId": 84532,
    "offset": 0
  }),
});
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface CommentsRequest {
  offset: number;
  moment: {
    contractAddress: string,
    tokenId: string
  };
  chainId: number;
}

const payload: CommentsRequest = {
  "moment": {
    "contractAddress": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
    "tokenId": "1"
  },
  "chainId": 84532,
  "offset": 0
};

const response = await fetch("https://inprocess.fun/api/moment/comments", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(payload),
});
const data = await response.json();
```
