# Get Airdrop

This page documents the API endpoint for retrieving Moment airdrop information on the In Process protocol.

## Endpoint

```http
GET https://inprocess.world/api/moment/airdrop
```

## Query Parameters

The following query parameters are available. Only `collectionAddress` and `tokenId` are required:

### Required query parameters

- `collectionAddress` (string): Moment contract address
- `tokenId` (string): Token ID

### Optional query parameters

- `chainId` (number): Chain ID (default: Base, 8453)
- `offset` (number): Pagination offset (default: 0)

## Example Requests

:::code-group

```bash [cURL]
curl "https://inprocess.world/api/moment/airdrop?collectionAddress=0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E&tokenId=1&chainId=84532&offset=0"
```

```python [Python]
import requests

url = "https://inprocess.world/api/moment/airdrop"
params = {
  "collectionAddress": "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
  "tokenId": "1",
  "chainId": 84532,
  "offset": 0
}
response = requests.get(url, params=params)
print(response.json())
```

```javascript [JavaScript]
const params = new URLSearchParams({
  collectionAddress: "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",

  offset: "0",

const response = await fetch(

```

```typescript [TypeScript]
interface AirdropQueryParams {
  collectionAddress: string;
  tokenId: string;
  chainId?: number;
  offset?: number;
}

const params: AirdropQueryParams = {
  collectionAddress: "0x297F21AAc502571db2c7dBFD271E68Dae7F2cE1E",
  tokenId: "1",
  chainId: 84532,
  offset: 0,
};

const queryString = new URLSearchParams({
  collectionAddress: params.collectionAddress,
  tokenId: params.tokenId,
  chainId: (params.chainId ?? 8453).toString(),
  offset: params.offset?.toString() || "0",
});

const response = await fetch(
  `https://inprocess.world/api/moment/airdrop?${queryString}`
);
const data = await response.json();
```
