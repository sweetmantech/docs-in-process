# Getting Started

Welcome to In Process! This guide will help you get started with our protocol.

## Quick Start

1. Visit our website at [https://www.inprocess.fun/](https://www.inprocess.fun/)
2. Sign in
3. Create your first moment through our user-friendly interface
4. Expand your collection by creating [additional moments/](https://in-process-docs.vercel.app/expand-collection)

For more detailed information about specific features, please refer to our other documentation sections.
