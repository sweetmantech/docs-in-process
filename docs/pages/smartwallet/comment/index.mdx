# Comment using Smart Wallet

This page documents API endpoints to mint free token using smart wallet in the In Process protocol.
No signature is required for this endpoint.

## Endpoint

```
POST https://inprocess.fun/api/smartwallet/comment
```

## Request Body

Send a JSON object with the following fields:

```json
{
  token: {
    tokenId: string; // free token id
    tokenContractAddress: string; // token contract address
  };
  account: string; // address to access a smart wallet
  to: string; // address to mint (artist wallet)
  comment: string; // comment
  amount: number; // amount to mint
}
```

## Example Request

:::code-group

```bash [cURL]
curl -X POST https://inprocess.fun/api/smartwallet/comment \
  -H "Content-Type: application/json" \
  -d '{
    {
      "account": "0x643D...8D01",
      "to": "0x5102...ad264",
      "token": {
          "tokenId": 1,
          "tokenContractAddress": "0xEE92...4D92"
      },
      "comment": "in process!!!",
      "amount": 1
    }
  }'
```

```python [Python]
import requests

url = "https://inprocess.fun/api/smartwallet/comment"
payload = {
  {
    "account": "0x643D...8D01",
    "to": "0x5102...ad264",
    "token": {
        "tokenId": 1,
        "tokenContractAddress": "0xEE92...4D92"
    },
    "comment": "in process!!!",
    "amount": 1
  }
}
response = requests.post(url, json=payload)
print(response.json())
```

```typescript [TypeScript]
interface mintCommentRequest {
  token: {
    tokenId: number;
    tokenContractAddress: string;
  };
  account: string;
  to: string;
  comment: string;
  amount: number;
}

const payload: mintCommentRequest = {
  "account": "0x643D...8D01",
  "to": "0x5102...ad264",
  "token": {
      "tokenId": 1,
      "tokenContractAddress": "0xEE92...4D92"
  },
  "comment": "in process!!!",
  "amount": 1
}

const response = await fetch("https://inprocess.fun/api/smartwallet/comment", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(payload),
});
const data = await response.json();
console.log(data);
```
