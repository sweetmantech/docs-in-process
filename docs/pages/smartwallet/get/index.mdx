# Getting a Smart Wallet

This page documents the API endpoint for creating a new Moment on the In Process protocol. No signature is required for this endpoint.

## Endpoint

```
POST https://inprocess.fun/api/smartwallet/get
```

## Request Body

Send a JSON object with the following fields:

```json
{
  "address": "string" // The address to access the smart wallet
}
```

## Example Request

:::code-group

```bash [cURL]
curl -X POST https://inprocess.fun/api/smartwallet/get \
  -H "Content-Type: application/json" \
  -d '{
    "address": "0x643d..8d01"
}'
```

```python [Python]
import requests

url = "https://inprocess.fun/api/smartwallet/get"
payload = {
    "address": "0x643d..8d01"
}
response = requests.post(url, json=payload)
print(response.json())
```

```javascript [JavaScript]
const response = await fetch("https://inprocess.fun/api/smartwallet/get", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    address: "0x643d..8d01"
  }),
});
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface GetSmartWalletRequest {
  address: string;
}
const payload: GetSmartWalletRequest = {
  address: "0x643d..8d01"
};
const response = await fetch("https://inprocess.fun/api/smartwallet/get", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(payload),
});
const data = await response.json();
console.log(data);
```

:::

## Example Response

```json
{
  "address": "0x41f1...3d51", // smart wallet address
}
```
