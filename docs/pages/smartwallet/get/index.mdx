# Getting a Smart Wallet

This page documents the API endpoint to get a smart wallet on the In Process protocol. No signature is required for this endpoint.

## Endpoint

```
GET https://inprocess.fun/api/smartwallet/get?address=0x643d..8d01
```

## Query Parameters

Send the address as a query parameter in the URL:

| Name      | Type   | Required | Default | Description                         |
| --------- | ------ | -------- | ------- | ----------------------------------- |
| address   | string | Yes      | â€”       | address to access a smart wallet    |


## Example Request

:::code-group

```bash [cURL]
curl -X GET "https://inprocess.fun/api/smartwallet/get?address=0x643d..8d01"
```

```python [Python]
import requests
params = {}
params["address"] = "0x643d..8d01"
url = "https://inprocess.fun/api/smartwallet/get"
response = requests.get(url, params=params)
print(response.json())
```

```javascript [JavaScript]
const params = new URLSearchParams();
params.set("address", "0x643d..8d01");
const response = await fetch(`https://inprocess.fun/api/smartwallet/get?${params}`);
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface GetSmartWalletResponse {
  address: string;
}
const params = new URLSearchParams();
params.set("address", "0x643d..8d01");
const response = await fetch(`https://inprocess.fun/api/smartwallet/get?${params}`);
const data: GetSmartWalletResponse = await response.json();
console.log(data);
```

:::
