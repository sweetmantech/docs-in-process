# Getting a Smart Wallet

This page documents the API endpoint for creating a new Moment on the In Process protocol. No signature is required for this endpoint.

## Endpoint

```
GET https://inprocess.fun/api/smartwallet/get?address=0x643d..8d01
```

## Query Parameters

Send the address as a query parameter in the URL:

```
?address=string // The address to access the smart wallet
```

## Example Request

:::code-group

```bash [cURL]
curl -X GET "https://inprocess.fun/api/smartwallet/get?address=0x643d..8d01" \
  -H "Content-Type: application/json"
```

```python [Python]
import requests

url = "https://inprocess.fun/api/smartwallet/get?address=0x643d..8d01"
response = requests.get(url)

print(response.json())
```

```javascript [JavaScript]
const address = "0x643d..8d01";
const response = await fetch(`https://inprocess.fun/api/smartwallet/get?address=${address}`, {
  method: "GET",
  headers: { "Content-Type": "application/json" },
});
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface GetSmartWalletResponse {
  address: string;
}
const address = "0x643d..8d01";
const response = await fetch(`https://inprocess.fun/api/smartwallet/get?address=${address}`, {
  method: "GET",
  headers: { "Content-Type": "application/json" },
});
const data: GetSmartWalletResponse = await response.json();
console.log(data);
```

:::

## Example Response

```json
{
  "address": "0x41f1...3d51", // smart wallet address
}
```
