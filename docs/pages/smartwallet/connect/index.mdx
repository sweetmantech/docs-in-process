# Connect a External Wallet

This page documents API endpoints for connecting external wallet with smart wallet in the In Process protocol.
No signature is required for this endpoint.

## Endpoint

```
GET https://inprocess.fun/api/smartwallet/connect
```

## Request Body

Send a JSON object with the following fields:

```json
{
  "smart_wallet": "0x1a4A...5F0A",
  "external_wallet": "0x364d...272e"
}
```

## Example Request

:::code-group

```bash [cURL]
curl -X POST https://inprocess.fun/api/smartwallet/connect \
  -H "Content-Type: application/json" \
  -d '{
    "smart_wallet": "0x1a4A...5F0A",
    "external_wallet": "0x364d...272e"
  }'
```

```python [Python]
import requests

url = "https://inprocess.fun/api/smartwallet/connect"
payload = {
  "smart_wallet": "0x1a4A...5F0A",
  "external_wallet": "0x364d...272e"
}
response = requests.post(url, json=payload)
print(response.json())
```

```javascript [JavaScript]
const response = await fetch("https://inprocess.fun/api/smartwallet/connect", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    "smart_wallet": "0x1a4A...5F0A",
    "external_wallet": "0x364d...272e"
  }),
});
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface ConnectExternalWalletRequest {
  smart_wallet: string;
  external_wallet: string;
}

const payload: ConnectExternalWalletRequest = {
 "smart_wallet": "0x1a4A...5F0A",
 "external_wallet": "0x364d...272e"
};

const response = await fetch("https://inprocess.fun/api/smartwallet/connect", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(payload),
});
const data = await response.json();
console.log(data);
```
