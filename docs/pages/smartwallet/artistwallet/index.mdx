# Getting an Artist Wallet

This page documents the API endpoint to get an artist wallet associated with smart wallet on the In Process protocol.

## Endpoint

```
GET https://inprocess.fun/api/smartwallet/artistwallet
```

## Query Parameters

Send the smart wallet address as a query parameter in the URL:

| Name          | Type   | Required | Default | Description                       |
| ------------  | ------ | -------- | ------- | --------------------------------- |
| smart_wallet  | string | Yes      | â€”       | The address of a smart wallet     |


## Example Request

:::code-group

```bash [cURL]
curl -X GET "https://inprocess.fun/api/smartwallet/artistwallet?smart_wallet=0xc6fc6bcae60cbb0ad41e21bb18b7f37b56da2571"
```

```python [Python]
import requests
params = {}
params["smart_wallet"] = "0xc6fc6bcae60cbb0ad41e21bb18b7f37b56da2571"
url = "https://inprocess.fun/api/smartwallet/artistwallet"
response = requests.get(url, params=params)
print(response.json())
```

```javascript [JavaScript]
const params = new URLSearchParams();
params.set("smart_wallet", "0xc6fc6bcae60cbb0ad41e21bb18b7f37b56da2571");
const response = await fetch(`https://inprocess.fun/api/smartwallet/artistwallet?${params}`);
const data = await response.json();
console.log(data);
```

```typescript [TypeScript]
interface GetArtistWalletResponse {
  address: string;
}
const params = new URLSearchParams();
params.set("smart_wallet", "0xc6fc6bcae60cbb0ad41e21bb18b7f37b56da2571");
const response = await fetch(`https://inprocess.fun/api/smartwallet/artistwallet?${params}`);
const data: GetArtistWalletResponse = await response.json();
console.log(data);
```

:::
